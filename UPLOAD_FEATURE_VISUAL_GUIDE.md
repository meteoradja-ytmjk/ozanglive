## Technical Details

### API Calls

1. **Open Modal**
   ```
   GET /api/render/jobs
   → Find job by ID
   ```

2. **Load Accounts**
   ```
   GET /api/youtube/accounts
   → Populate dropdown
   ```

3. **Load Defaults**
   ```
   GET /api/youtube/channel/:id/defaults
   → Display in info box
   ```

4. **Upload Video**
   ```
   POST /api/render/jobs/:id/upload
   Body: {
     targetAccountId: "123",
     title: "My Renamed Video Title",
     useChannelDefaults: true
   }
   → Upload to YouTube with defaults
   ```

### Data Flow

```
User Action → Frontend → Backend → YouTube API
    ↓            ↓          ↓           ↓
  Click      Validate   Fetch      Upload
  Upload     Input      Defaults   Video
    ↓            ↓          ↓           ↓
  Modal      Show       Apply      Return
  Opens      Progress   Metadata   Video ID
    ↓            ↓          ↓           ↓
  Select     Simulate   Upload     Update
  Channel    Progress   File       Database
    ↓            ↓          ↓           ↓
  Load       Update     Process    Show
  Defaults   UI         Video      Success
```
