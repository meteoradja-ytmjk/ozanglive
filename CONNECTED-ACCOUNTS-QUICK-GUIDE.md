# 🚀 Connected Accounts - Quick User Guide

## 📖 Apa yang Baru?

Sekarang daftar **Connected Accounts** bisa di-collapse/expand untuk menghemat ruang layar!

---

## 🎯 Cara Menggunakan

### 1️⃣ Melihat Jumlah Accounts
```
┌─────────────────────────────────────────┐
│ 👥 Connected Accounts (3)   [+ Add] ▶  │
│ Click to expand/collapse                │
└─────────────────────────────────────────┘
```
- Lihat badge **(3)** untuk tahu jumlah accounts
- Tidak perlu expand untuk info ini

### 2️⃣ Expand Daftar Accounts
**Klik pada header "Connected Accounts"**
```
┌─────────────────────────────────────────┐
│ 👥 Connected Accounts (3)   [+ Add] ▼  │ ← Chevron berubah
│ Click to expand/collapse                │
├─────────────────────────────────────────┤
│ 📺 Channel 1  [Primary] ⭐ ✏️ 🔗      │
│ 📺 Channel 2            ⭐ ✏️ 🔗      │
│ 📺 Channel 3            ⭐ ✏️ 🔗      │
└─────────────────────────────────────────┘
```

### 3️⃣ Collapse Daftar Accounts
**Klik lagi pada header**
```
┌─────────────────────────────────────────┐
│ 👥 Connected Accounts (3)   [+ Add] ▶  │ ← Chevron kembali
│ Click to expand/collapse                │
└─────────────────────────────────────────┘
```

### 4️⃣ Menambah Account Baru
**Klik button "Add Account"**
- Bisa diklik kapan saja (collapsed atau expanded)
- Modal akan terbuka
- Daftar accounts tidak akan expand otomatis

## ❓ FAQ

### Q: Kenapa default collapsed?
**A:** Untuk menghemat ruang layar dan fokus ke broadcasts (konten utama).

### Q: Apakah data accounts hilang saat collapsed?
**A:** Tidak! Data tetap ada, hanya tersembunyi. Klik header untuk expand.

### Q: Bagaimana cara add account tanpa expand?
**A:** Klik button "Add Account" langsung, tidak perlu expand dulu.

### Q: Apakah bisa expand multiple sections?
**A:** Ya! Accounts dan broadcasts bisa di-expand bersamaan.

### Q: Apakah state tersimpan saat reload?
**A:** Saat ini belum, tapi fitur ini akan ditambahkan di update berikutnya.

## 🔧 Troubleshooting

### Masalah: Chevron tidak berubah
**Solusi:** Refresh halaman (F5)

### Masalah: Accounts tidak muncul saat expand
**Solusi:** 
1. Check console untuk errors (F12)
2. Refresh halaman
3. Clear browser cache

### Masalah: Button "Add Account" tidak berfungsi
**Solusi:**
1. Pastikan tidak ada modal lain yang terbuka
2. Refresh halaman
3. Check browser console
