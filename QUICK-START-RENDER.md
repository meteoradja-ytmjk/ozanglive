# Quick Start Guide - Render Dashboard

## 🚀 How to Use the New Features

### 1. Upload Video/Audio Files

**Step 1:** Go to Render Dashboard → Step 3 (Select Videos & Audio)

**Step 2:** Click the **Upload** button (green icon)

**Step 3:** Choose upload type:
- Click **Video Files** for videos
- Click **Audio Files** for audio

**Step 4:** Add files:
- **Drag & Drop** files into the drop zone, OR
- Click **Browse Files** to select from your computer

**Step 5:** Click **Upload Files**
- Multiple files upload in parallel
- Progress bar shows upload status
- Files appear in gallery immediately

**Tips:**
- ✅ Upload multiple files at once
- ✅ Files are available immediately after upload
- ✅ No need to refresh the page

---

### 2. Set Video Duration

**Step 1:** Go to Step 2 (Duration Settings)

**Option A: Manual Duration**
- Set **Hours** (0-99)
- Set **Minutes** (0-59)
- Example: 5 hours 30 minutes

**Option B: Follow Audio Duration**
- Check the **Follow Audio** checkbox
- Duration will automatically match total audio length
- Perfect for music videos!

**Option C: Mute Video Audio** 🆕
- Check the **Mute Video** checkbox
- Removes original audio from video
- Use cases:
  - Replace video audio with your own music
  - Create silent videos
  - Remove unwanted background noise

**Tips:**
- ✅ Duration is now accurate
- ✅ Follow Audio works perfectly
- ✅ Mute Video removes original audio
- ✅ No more 1:21 bug!

---

### 3. Schedule Render Jobs

**Step 1:** Configure your render job:
- Add title
- Select channel (optional)
- Set duration
- Select videos and audio

**Step 2:** Click **Schedule** button (purple, Step 4)

**Step 3:** Choose scheduling mode:

**Mode 1: Date & Time**
- Select specific date
- Select specific time
- See preview of scheduled time

**Mode 2: Delay Duration**
- Set hours delay (0-99)
- Set minutes delay (0-59)
- See preview of when job will run

**Step 4:** Click **Schedule Render**
- Job will run automatically at scheduled time
- No need to stay online!

**Tips:**
- ✅ Schedule jobs for off-peak hours
- ✅ Schedule multiple jobs in advance
- ✅ Time is shown in your local timezone

---

### 4. Render Speed Optimization

The system automatically chooses the fastest method:

#### ⚡⚡⚡ ULTRA FAST (5-15 seconds)
**When:** 1 video + 1 audio
**Speed:** 30x faster than before
**Example:** 5-minute video renders in 10 seconds

#### ⚡⚡ FAST (15-35 seconds)
**When:** 1 video + multiple audios
**Speed:** 5-10x faster than before
**Example:** 1 video + 5 audios renders in 25 seconds

#### 📦 STANDARD (1-3 minutes)
**When:** Multiple videos + audios
**Speed:** Reliable and stable
**Example:** 10 videos + 10 audios renders in 2 minutes

**Tips:**
- ✅ System chooses best method automatically
- ✅ No configuration needed
- ✅ Progress bar shows real-time status

## ❓ Troubleshooting

### Upload Issues:
**Problem:** Upload fails
- ✅ Check internet connection
- ✅ Check file size (max 2GB per file)
- ✅ Check file format (MP4, AVI, MOV, MKV, WebM)

**Problem:** Files not appearing
- ✅ Wait for upload to complete
- ✅ Check progress bar
- ✅ Refresh page if needed

### Duration Issues:
**Problem:** Duration not working
- ✅ Make sure you set hours OR minutes
- ✅ If using Follow Audio, select audio files first
- ✅ Check that audio files are valid

### Render Issues:
**Problem:** Render is slow
- ✅ Check if using multiple videos (slower but reliable)
- ✅ Check system resources (CPU, RAM)
- ✅ Schedule for off-peak hours

**Problem:** Render fails
- ✅ Check that all files exist
- ✅ Check file formats are supported
- ✅ Check logs for error details
